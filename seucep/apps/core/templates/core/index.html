Opções

Você pode formatar a saída usando o header Accept: application/json; indent=4 para formatar a saída

Formato YAML

Para receber a saída no formato yaml basta usar Accept: application/yaml para formatar a saída

Formato Piped

Para receber a como um pipe basta usar Accept: text/piped para formatar a saída

Formato XML

Para receber a como um XML basta usar Accept: application/xml para formatar a saída
